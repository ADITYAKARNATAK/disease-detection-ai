# Disease Detection AI Web App

## Project Description

This project is a Machine Learning based Disease Detection Web Application.

The system predicts disease based on:

- Symptoms input
- Medical report text
- Uploaded report file

It uses Machine Learning and Flask to create a real web application.


## Features

- Disease prediction using symptoms
- Prediction using report text
- File upload (PDF / Image / TXT)
- OCR support
- Chatbot style input
- Flask backend
- Bootstrap UI
- SQLite database history
- Deployment ready
- GitHub ready


## Technologies Used

- Python
- Flask
- Machine Learning
- Pandas
- NumPy
- Scikit-learn
- SQLite
- HTML
- CSS
- Bootstrap


## Dataset

Disease prediction dataset from Kaggle.

Contains symptoms and diseases.


## How to Run Locally

```
git clone <repo link>
cd disease-detection-ai

pip install -r requirements.txt

python app.py
```

Open:

```
http://127.0.0.1:5000
```
